# Netflix Movies & TV Shows — Exploratory Data Analysis

## Overview
An exploratory data analysis of the Netflix content library 
to uncover trends in content type, genre, country, growth, 
and audience ratings.

## Questions Explored
- What is the split between Movies and TV Shows?
- Which countries produce the most content?
- What are the most popular genres?
- How has Netflix's content library grown over the years?
- What audience does Netflix primarily target based on ratings?

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Netflix Titles Dataset from Kaggle
