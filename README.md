# Netflix Movies & TV Shows: Exploratory Data Analysis

## Overview
An exploratory data analysis of the Netflix content library 
to uncover trends in content type, genre, country, growth, 
and audience ratings.

## Questions Explored
1. What is the split between Movies and TV Shows?
2. Which countries produce the most content?
3. What are the most popular genres?
4. How has Netflix's content library grown over the years?
5. What audience does Netflix primarily target based on ratings?

## Tools Used
1. Python
2. Pandas
3. Matplotlib
4. Seaborn
5. Jupyter Notebook

## Dataset
Netflix Titles Dataset from Kaggle
